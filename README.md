# Enterprise RAG Platform

An enterprise-grade AI Knowledge Platform built using LangChain, FastAPI, and modern Retrieval-Augmented Generation (RAG) techniques.

## Project Goals

- Build a production-ready RAG system
- Implement advanced retrieval strategies
- Develop an Agentic RAG workflow
- Deploy the application to the cloud
- Evaluate and optimize retrieval quality

## Tech Stack

- Python
- FastAPI
- LangChain
- LangGraph
- ChromaDB
- Hugging Face Embeddings
- Streamlit
- Docker
